


# Test Plan - Krakvet.pl

## 1. Introduction

Document describes the testing approach for selected functionalities of the Krakvet.pl pet online shop

## 2. Objectives

- Verify correctness of the user registration process
- Identify usability and communication issues
- Ensure the user receives clear feedback after critical actions

## 3. Scope of testing

### In scope

- User registration flow
- Password setup process
- User feedback and system messages
- Login after registration

## 4. Test Type
- Manual testing
- Exploratory testing
- Functional testing
- UX/UI testing

## 5. Test Environment
- Website: https://www.krakvet.pl
- Browser: Google Chrome
- OS: Windows 11
- Test type: Production (observational testing)
